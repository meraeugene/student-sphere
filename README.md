> A Student Portal built with the React and PHP.
>
> It is a full-featured student portal system with admin, teacher, and student users.
>
>
> ## Features

ADMIN FEATURES:
- Admin Dashboard
- Faculty Management
- Student Management
- Department Management
- Program Management
- Subject Management
- Section Management
- Assign Subjects

FACULTY FEATURES:
- Faculty Dashboard
- Grade Management

STUDENT FEATURES:
- Enrolled Subjects
- Report of Grades

ALL USERS FEATURES:
- Profile Mangament
- Password & Security (Reset Password)
